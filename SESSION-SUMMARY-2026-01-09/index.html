    Architecture Pattern

    Hybrid Mono-Repo with Plugin-Scoped Documentation:

    claude-plugins/
├── docs/
│   ├── plugins/                    # Landing pages
│   │   ├── index.md
│   │   ├── rforge.md
│   │   ├── craft.md
│   │   ├── workflow.md
│   │   └── statistical-research.md
│   ├── rforge/docs/                # Copied from plugin
│   ├── craft/docs/
│   ├── workflow/docs/
│   └── statistical-research/docs/
├── rforge/docs/                    # Source of truth
├── craft/docs/
├── workflow/docs/
└── statistical-research/docs/

    Benefits: - Plugin docs stay with plugin code (maintainability) - Unified documentation site (user experience) - Clear source of truth (plugin directories) - Easy deployment (copy to docs/)
