    First Commands

    Progress: Step 3 of 4

    Trying essential commands - takes 20 seconds

    Universal Task Execution

    /craft:do "add user authentication"

    The AI routes your task to the best workflow automatically.

    Quick Win: Try It Now

    Run /craft:do "show me what you can do" - the AI will analyze your project and suggest the most useful commands for your current context.

    Pre-Flight Checks

    /craft:check                    # Quick validation
/craft:check --for release      # Full release audit

    Pro Tip: Run Before Every Commit

    Make /craft:check part of your commit workflow - it catches issues in seconds and prevents broken commits.

    Context-Aware Help

    /craft:help                     # Suggestions for your project
/craft:help testing             # Deep dive into testing

    Common Workflows

    Documentation Workflow

    # 1. Update docs from code changes
/craft:docs:update

# 2. Check for issues
/craft:docs:check

# 3. Deploy
/craft:site:deploy

    Release Workflow

    # 1. Pre-release audit
/craft:check --for release

# 2. Generate changelog
/craft:docs:changelog

# 3. Git cleanup
/craft:git:clean

    Development Workflow

    # 1. Create feature branch with worktree
/craft:git:worktree add feature-name

# 2. Run tests in watch mode
/craft:test:run debug

# 3. Lint with auto-fix
/craft:code:lint optimize
