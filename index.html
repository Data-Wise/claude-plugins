    RForge

    Auto-delegation orchestrator for R package development with explicit mode control

    • 4 analysis modes - Explicit control over analysis depth and time
    • Intelligent delegation - Automatically selects appropriate RForge MCP tools
    • Performance guarantees - Default mode < 10s (MUST), others up to 5m
    • Format options - Terminal, JSON, or Markdown output
    • 100% backward compatible - All existing commands work unchanged

    Analysis Modes: - Default - Fast, balanced analysis (< 10 seconds) - Use for daily development - Debug - Deep inspection with detailed traces (< 2 minutes) - Use for troubleshooting - Optimize - Performance-focused analysis (< 3 minutes) - Use for bottleneck identification - Release - Comprehensive CRAN validation (< 5 minutes) - Use before package release

    Commands: - /rforge:analyze [mode] - Flexible analysis with optional mode (default, debug, optimize, release) - /rforge:status [mode] - Package status dashboard with optional detail level - /rforge:quick - Ultra-fast convenience command (equivalent to analyze default) - /rforge:thorough - Comprehensive convenience command (equivalent to analyze release)

    Statistical Research

    Research workflow automation for statistical methods development

    • 14 commands - Literature search, manuscript writing, simulation design, method scouting
    • 17 A-grade skills - Mathematical foundations, implementation, writing, research
    • MCP integration - Connects to arXiv, Zotero, R environment
    • Publication-ready - Methods papers, proofs, reviewer responses

    Command Categories: - Literature - arXiv search, DOI lookup, BibTeX management - Manuscript - Methods sections, results, reviewer responses, proofs - Simulation - Study design, analysis planning - Research - Literature gaps, hypothesis formulation, analysis plans, method scouting

    • Auto-delegation - Automatically delegates to specialized agents

    • Design modes - Technical, creative, user-focused, system-wide

    Command Categories: - Core Workflow - brainstorm, focus, next, done, recap - Problem Solving - stuck, refine - Task Management - task-status, task-output, task-cancel

    Getting Started

    1. Installation
    # Clone repository
git clone https://github.com/Data-Wise/claude-plugins.git
cd claude-plugins

# Install a plugin
./scripts/install-plugin.sh rforge
    1. Restart Claude Code

    2. Commands won't appear until you restart

    3. Use Commands

    /rforge:quick
/research:arxiv "mediation analysis"
/brainstorm "API design"
